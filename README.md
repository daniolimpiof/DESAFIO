
# ✨Desafio Técnico Dot Digital Group✨

Esta aplicação consiste em uma landing page responsiva desenvolvida com objetivo de demonstrar competências em front-end para o desafio técnico da empresa Dot Group.


## 🚀Stack utilizada

**Front-end:** HTML, CSS, Javascript.



## 🛣️Roadmap

- Desenvolvimento e organização do corpo HTML.

- Implementação da lógica referente ao dropdown.

- Implementação da lógica referente ao círculo interativo.

- Estilização das páginas e dos componentes.

- Responsividade para despositivos entre 360px X 1920px.
